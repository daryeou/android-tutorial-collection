## Introduce

Kotlin 기반의 안드로이드 앱을 연습하기 위해 만든 예시 앱입니다.</br>
Activity와 Layout간 ViewBinding이 적용되었습니다.</br>
디자인 패턴은 적용하지 않았습니다.</br>
